The use my assembler:

1. Ensure that you have access to linux.
2. Open the linux command line.
2. Clone the the Assembler repository remotely.
3. Navigate to the repository directory.
4. Type 'make'
5. The code is now compiled. Next, type './project2 <filename>', where filename is one of the many .sic files
in the directory. 
6. A .obj file corresponding to the sic file you just assembled should now be present in the project directory.
